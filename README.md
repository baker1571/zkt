A simple CLI interface for managing a Zettelkasten note-taking system.

Although there are already CLI programs out there for managing a Zettelkasten like Vimwiki, I wanted to create a custom program that would provide a specific kind of control over the project without including features that I don’t need. 

This script searches through a designated Zettelkasten folder, recursively collecting all files with a 14-digit identifier prefix (yyyymmddhhmmss) and displaying them in a TUI.
